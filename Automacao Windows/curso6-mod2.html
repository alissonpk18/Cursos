Como Fazer o Pedido (O Prompt)

Pedir apenas "crie um programa de limpeza" é perigoso. Veja a estrutura de 4 partes para um pedido funcional e seguro.

⏱️ 6 min de leitura 🛡️ Segurança em 4 Pilares

O Perigo do Pedido Simples

Se você abrir o Gemini ou outra IA e digitar apenas: "crie um programa para limpar o PC", a IA provavelmente devolverá um código genérico.

Esse código genérico pode tentar apagar arquivos importantes do Windows, tentar forçar a exclusão de arquivos vitais que o sistema está usando no momento (gerando travamentos), ou executar sem as permissões adequadas, falhando silenciosamente.

Um pedido de automação seguro e funcional precisa seguir uma receita de quatro partes obrigatórias.

As 4 Partes de um Prompt de Automação

Toda vez que você pedir um script para gerenciar seu computador, inclua estes quatro pilares:

  1. Ação Direta Diga exatamente o que você quer resolver e em quais diretórios/locais a IA deve atuar. Seja específico.
  2. Exigência de Permissão (Trava de Segurança) Exija que o script verifique se foi aberto como Administrador. Se não foi, o script deve mostrar um aviso e fechar sozinho sem fazer nada. Isso evita que ele falhe no meio do caminho por falta de acesso.
  3. Proteção contra Travamentos (Ignorar Falhas) Ordene à IA que coloque comandos para "ignorar falhas" ou "seguir em frente silenciosamente" se um arquivo não puder ser apagado. O Windows sempre mantém alguns arquivos abertos em segundo plano, e tentar forçar a exclusão deles gera erros vermelhos e paralisa o programa.
  4. Comunicação Clara (Mensagens Coloridas) Exija que o script avise o que está fazendo em português, usando cores (exemplo: verde para tarefas concluídas com sucesso, amarelo para avisos e vermelho para erros graves).

Antes e Depois na prática

❌ Prompt Inseguro

"Escreva um script para limpar os arquivos temporários do meu Windows."

Resultado: Script sem trava de Administrador, que gera dezenas de erros vermelhos na tela ao tentar apagar arquivos em uso pelo Windows.
✓ Prompt Seguro (Com os 4 Pilares)

"Crie um script em Ferramenta de Comandos do Windows que limpe arquivos temporários. Verifique permissão de Administrador no início e pare se não tiver. Ignore arquivos em uso silenciosamente para não travar, e mostre mensagens verdes a cada pasta limpa."

Resultado: Execução limpa, avisa quando precisa ser aberto como administrador, roda sem erros na tela e é 100% legível.
