Teste, Execução e Correção

Agora você tem o programa salvo na Área de Trabalho. A etapa de testes garante que ele funcionará sempre com total segurança.

⏱️ 8 min de teste ⚙️ Execução Prática

1. O Teste da Trava de Segurança

Antes de rodar o programa de verdade, precisamos garantir que as travas de segurança estão ativas:

🔒
Dê um clique duplo normal no arquivo `Limpeza.ps1`.

Se o prompt foi feito corretamente, uma janela preta deve se abrir rapidamente, mostrar uma mensagem em vermelho avisando que você não tem permissões de Administrador e fechar sozinha. Isso prova que a trava de segurança funciona e o script não executará de forma incompleta.

2. A Execução Correta

Com a trava validada, agora podemos rodar o programa com as permissões corretas de manutenção:

  1. Clique com o botão direito: Clique com o botão direito do mouse em cima do arquivo Limpeza.ps1.
  2. Selecione o Ferramenta de Comandos do Windows: No menu que abrir, clique em Executar com o Ferramenta de Comandos do Windows (ou, dependendo das configurações do seu Windows 11, escolha "Abrir como Administrador").
  3. Observe a tela: Uma tela azul ou preta do terminal se abrirá. Acompanhe as mensagens que você pediu para a IA colocar. Se aparecerem mensagens coloridas e verdes informando as pastas limpas, o programa executou com sucesso.

3. Como Corrigir Erros

Caso apareça alguma mensagem de erro do sistema (letras vermelhas com termos técnicos) ou o programa pare no meio:

🛠️
Siga a receita de correção com a IA:
  1. Copie o texto vermelho de erro exibido na tela do Ferramenta de Comandos do Windows.
  2. Cole de volta na mesma conversa com a IA (Gemini ou Antigravity).
  3. Envie esta mensagem: "Ao rodar o script no meu Windows 11, deu este erro: [cole o erro]. Corrija o script para que isso não aconteça."
  4. A IA corrigirá o script. Copie a nova resposta, salve por cima do arquivo antigo no Bloco de Notas e repita o teste.

Concluiu os testes e a execução com sucesso? Marque o módulo como concluído.
