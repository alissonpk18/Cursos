O Projeto Prático (Criando o Pedido)

Neste módulo, você aplicará as regras do Módulo 2 para pedir um programa de limpeza de lixo digital (arquivos temporários).

⏱️ 8 min de prática 📋 Cópia e Envio do Prompt

O Texto do Pedido

Abaixo está o texto exato estruturado com as 4 regras de segurança para você enviar à IA (como Gemini ou Antigravity). Copie e cole na conversa.

Atue como um especialista em Windows 11. Escreva um script em Ferramenta de Comandos do Windows para limpar o lixo digital do computador. Não me explique o código, apenas entregue o script pronto. Siga estas regras obrigatórias: 1. O script deve verificar se foi aberto como Administrador. Se não foi, deve exibir um aviso vermelho e fechar sozinho. 2. O script deve esvaziar as quatro pastas principais de arquivos temporários do Windows. 3. O script deve ter regras para 'ignorar em silêncio' qualquer arquivo que o Windows estiver usando no momento, para não causar travamentos na tela negra. 4. O script deve pausar as atualizações do Windows antes de limpar, e religá-las no final. 5. Mostre mensagens na tela em português dizendo o que está acontecendo em cada etapa.

Ao clicar no botão acima, o prompt será copiado para sua área de transferência. Abra a IA em uma nova aba e envie a mensagem.

O que esperar da IA?

A Inteligência Artificial deve responder com um bloco de código (com fundo cinza escuro ou preto) contendo comandos do Ferramenta de Comandos do Windows, que começam com algo parecido com:

# Testando permissões de administrador
$isAdmin = ([Security.Principal.WindowsPrincipal][Security.Principal.WindowsIdentity]::GetCurrent()).IsInRole([Security.Principal.WindowsBuiltInRole]::Administrator)
if (-not $isAdmin) {
    Write-Host "ERRO: Rode como Administrador!" -ForegroundColor Red
    ...
}

Não se preocupe em entender a sintaxe. O importante é que a IA obedeceu às regras e colocou a trava de segurança logo no início.
